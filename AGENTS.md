## Project Summary
Secure City IQ is a high-tech urban intelligence dashboard designed for modern, data-driven city management. It features a futuristic cyber-aesthetic with animated backgrounds and glassmorphic UI elements.

## Tech Stack
- Framework: Next.js 15
- Styling: Tailwind CSS
- Animation: Custom CSS Animations
- Icons: Lucide (via Next.js Image)

## Architecture
- `src/app/page.tsx`: Main dashboard landing page.
- `src/app/globals.css`: Custom cyber-grid animations and glassmorphism utilities.

## User Preferences
- Theme: High-tech Cyber (Dark Teal/Cyan)
- UI Style: Glassmorphism with neon glow effects
- Layout: Perfectly centered and professional

## Project Guidelines
- Use glassmorphism for all cards and buttons.
- Maintain a cohesive Dark Teal/Cyan color scheme.
- Ensure all interactive elements have hover scale and glow effects.

## Common Patterns
- Glassmorphic components: `glass-card`, `glass-button`.
- Typography: `neon-text` for headings.
